# Evolve

[![License: ISC](https://img.shields.io/badge/License-ISC-blue.svg)](https://opensource.org/licenses/isc)

My basic Neovim setup based heavily on info found in a You Tube video.

I have just picked the parts I want to use and chosen a different theming approach, as I want it to auto-adapt to a dark or light terminal on its own.

This repo was created so I have easy access to the settings. 

## install

Clone the repo into a subfolder in `~/.config/nvim/lua/` and load it using the `~/.config/nvim/init.lua` file.

```lua
require("evolve")
```
